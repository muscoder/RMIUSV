-Install bcm2835 library for programming GPIO of Raspberry PI with C programming language

Installation Instructions:
- extract the files from bcm2835-1.36.tar.gz
  On terminal:
     tar -xzvf bcm2835-1.36.tar.gz
- change current directory to extracted directory
  On terminal:
     cd bcm2835-1.36
- apply below commands respectively
  On terminal:
     ./configure
     make
     sudo make check
     sudo make install
